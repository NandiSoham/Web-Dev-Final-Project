This ML project aims to detect lung tuberculosis using machine learning algorithms. Soccer Management System

This project focuses on developing a soccer management system using a dataset containing information about goal details, player details, coach, and referee details. The system will provide an organized and well-documented approach to the game, keeping track of statistics on teams, players, events, and more. The database includes tables for goal details, match details, penalty shootout, player in-out, and others. The system will be designed to handle data related to various types of goals, substituted players, missed goals, win-lose or draw status, goal difference, and more. Overall, the project aims to create an efficient and user-friendly platform for managing and analyzing soccer-related data.
